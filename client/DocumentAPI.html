Creates a pokemon /PostPokemons

Body Params: Name type: string - Name of the Pokemon, Type type: string - Type of the Pokemon, ID type: integer - ID of the Pokemon

Method: POST

Description: This endpoint creates a Pokemon by with their name, type and id. It returns a JSON object containing the Pokemon's data.

Example Response:

Request: /PostPokemons

Result:

{   
    "pokemon": [
        {
            "id": 152,
            "num": "152",
            "name": "Chikorita",
            "type": [
            "Grass"
            ],
        ]
    }
}

Request: /PostPokemons (If the user doesn't provide all required fields)

Result:

{   
   404 error: Pokemon can't be created without a name, type and id 
}
