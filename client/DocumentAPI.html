Documentation

Documentation for the Project. What I have for the API endpoints and how to use them.

Endpoints

  • GET /GetAll: Returns a list of all Pokemon in the database.
  • GET /GetName?name=NAME: Returns the Pokemon with the specified name.
  • GET /GetHeight?height=HEIGHT: Returns a list of Pokemon with the specified height.
  • GET /GetWeight?weight=WEIGHT: Returns a list of Pokemon with the specified weight.
  • POST /Add: Adds a new Pokemon to the database. Requires a JSON body with the following format: 
            {
            "name": "NAME",
            "height": HEIGHT,
            "weight": WEIGHT
        }
  • PUT /Update: Updates an existing Pokemon in the database. Requires a JSON body with the following format: 
            {
            "name": "NAME",
            "height": HEIGHT,
            "weight": WEIGHT
        }
  • DELETE /Delete?name=NAME: Deletes the Pokemon with the specified name from the database.
